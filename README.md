# Jami Configuration Files

This repository hosts configuration files for self-hosted Jami infrastructure.

## Proxy List

JSON file with DHT proxy servers:
- **URL**: `https://avvdev.github.io/proxy_list.json`

### Usage in Jami

**Desktop (Windows/Linux/macOS):**
1. Settings → Account → Advanced settings
2. Proxy list URL: `https://avvdev.github.io/proxy_list.json`

**Android:**
1. Settings → Account → [Your account] → Advanced
2. DHT proxy list URL: `https://avvdev.github.io/proxy_list.json`
